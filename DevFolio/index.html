PokeDex

The Pokédex was created by Professor Oak as an invaluable tool for trainers in the Pokémon world. Lists information about every Pokémon in the world that is contained in the database.

This pokedex is based on API technology. I used PokeApi here that provides all the necessary information about a given pokemon.

This Pokédex contains all the first generation Pokémon.
